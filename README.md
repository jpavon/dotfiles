# Dotfiles

## Usage:

#### Clone this repo:

```bash
git clone https://github.com/jpavon/dotfiles.git ~/.dotfiles
```

#### Make sync.sh executable:

```bash
cd ~/.dotfiles && chmod +x sync.sh
```

#### Sync files:

```bash
./sync.sh
```

Once is synced you can use this commands:

```bash
dot # sync dotfiles
godot # go to ~/.dotfiles folder
```
